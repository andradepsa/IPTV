🔧 COMO FUNCIONA O PLAYER
Fluxo de reprodução (Archive.org):
1. Busca metadados: GET archive.org/metadata/{identifier}
2. Extrai a melhor URL de vídeo dos files[] (prioridade: MPEG4 → H.264 → HLS)
3. Toca diretamente no <video> via HLS.js (browsers sem suporte nativo) ou nativo (Safari/iOS)
4. Fallback automático: se o metadata falhar → tenta via proxies CORS → embed iframe como último recurso
