🔧 Como funciona
Busca e reprodução:
1. Consulta ao Archive.org via JSONP (sem CORS)
2. Para cada item, extraímos a URL real do vídeo (prioridade MPEG4 → H.264 → HLS)
3. Player HTML5 com suporte a HLS.js para formatos .m3u8
4. Fallback automático: embed iframe do Archive.org caso falhe
