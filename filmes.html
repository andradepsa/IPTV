💡 DICA DE INTEGRAÇÃO — COMO USAR NO SEU PROJETO
Estratégia recomendada: Use o Internet Archive para filmes completos gratuitos (já integrado), o Jikan para descoberta de anime, e o TMDB para enriquecer qualquer filme com poster, sinopse e nota. Para saber onde assistir, combine com Watchmode.

TMDB é a melhor API gratuita: Crie uma conta em themoviedb.org → Configurações → API → "Solicitar chave" (uso pessoal). Em menos de 2 minutos você tem acesso ilimitado a 700k+ filmes, arte em alta resolução e endpoints de documentários (genre_id=99), animação (genre_id=16), sci-fi (genre_id=878) e mais.
